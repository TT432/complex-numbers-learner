# 复数探索 · 交互式学习 🧮

通过拖拽、观察和练习，直观理解复数的几何意义。

## 章节

| # | 章节 | 内容 |
|---|------|------|
| 1 | 认识复数 | 复平面、实部/虚部 |
| 2 | 复数的表示 | 直角坐标 ↔ 极坐标 |
| 3 | 加减法 | 向量的平移 |
| 4 | 乘除法 | 旋转 + 缩放 |
| 5 | 共轭与模 | 关于实轴对称 |
| 6 | De Moivre 定理 | 乘方与单位根 |
| 7 | 综合练习 | 随机出题测验 |

## 技术栈

- 纯前端 SPA（无构建、无依赖）
- HTML5 Canvas + JS 复平面引擎
- KaTeX 公式渲染（CDN）
- 暗色主题、响应式布局（支持手机触屏拖拽）
- 部署于 GitHub Pages

## 本地运行

```bash
# 任意静态服务器
python3 -m http.server 8080
# 或
npx serve
```
